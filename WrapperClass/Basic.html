Wrapper Class in Java

Primitive datatype ka hi Wrapper class use hota hai. Jaha primitive datatype directly use nahi kar sakte, waha Wrapper class use karte hain.

Primitive Datatype → Wrapper Class

  1. byte → Byte
  2. short → Short
  3. int → Integer
  4. long → Long
  5. float → Float
  6. double → Double
  7. boolean → Boolean
  8. char → Character

Memory Concept

  • Primitive data Stack memory mein store hota hai.
  • Wrapper class object Heap memory mein store hota hai.
  • Wrapper variable ka reference Stack mein hota hai.

Important Concepts

  • Auto Boxing: Primitive ko Wrapper class mein convert karna.
  • Unboxing: Wrapper class ko primitive mein convert karna.
  • == reference compare karta hai.
  • .equals() value compare karta hai.
  • Integer caching range: -128 to 127.
